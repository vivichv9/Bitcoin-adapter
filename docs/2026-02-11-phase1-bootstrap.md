# Документация изменений от 2026-02-11 (этап 1)

## Что реализовано
1. Инициализирован JavaScript веб-проект:
- добавлен `package.json` со скриптами `dev`, `build`, `test`, `lint`, `check`.

2. Создан каркас UI:
- `index.html` и `styles.css`;
- форма конвертации в hex;
- блок ошибок и блок вывода hex;
- секция отображения script-типов по публичному ключу.

3. Реализована базовая логика приложения:
- `src/lib/tx.js`: генерация unsigned raw tx hex (1 input / 1 output), OP_RETURN script hex;
- `src/lib/validation.js`: валидация формы raw tx и public key;
- `src/lib/script-types.js`: шаблоны возможных script-типов по pubkey;
- `src/main.js`: обработчики форм, вывод результата и пример `bitcoin-cli` команд.

4. Добавлены инструменты разработки без внешних зависимостей:
- `scripts/dev-server.mjs`: локальный статический dev-сервер;
- `scripts/build.mjs`: сборка в `dist` + синтаксическая проверка JS;
- `scripts/lint.mjs`: синтаксические проверки исходников.

5. Добавлены тесты:
- `tests/tx.test.mjs` для проверок генерации raw tx и OP_RETURN.

6. Обновлен `README.md`:
- добавлены команды запуска/сборки/проверки;
- обновлен статус проекта.

## Ограничения текущей реализации
1. Raw tx генерируется как unsigned шаблон и не подписывается.
2. Для script-типов по pubkey отображаются шаблоны и дескрипторы, а не полные вычисленные адреса.
3. Генерация адресов и проверка подписи транзакций будут реализованы на следующих этапах плана.
