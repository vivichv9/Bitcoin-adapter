# Документация изменений от 2026-02-11 (.gitignore)

## Что изменено
1. Создан файл `.gitignore`.
2. Добавлены исключения для:
- зависимостей (`node_modules/`);
- артефактов сборки (`dist/`, `build/`, `coverage/`, кеши);
- логов (`*.log`, npm/yarn/pnpm debug logs);
- локальных env-файлов (`.env`, `.env.*`, кроме `.env.example`);
- файлов IDE и ОС (`.idea/`, `.vscode/`, `.DS_Store`, `Thumbs.db`).

## Зачем
Файл исключает служебные и локальные файлы из git, чтобы в репозиторий попадал только исходный код и проектная документация.
