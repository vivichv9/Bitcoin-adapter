# Документация изменений от 2026-02-11 (UX layout)

## Что изменено
1. Полностью переработан layout в `index.html`:
- левая колонка: заполнение транзакции (форма + итоговый hex + cli пример);
- правая колонка:
  - верх: кнопки и формы для взаимодействия с блокчейном;
  - низ: история запросов.

2. Добавлена компактная раскрывающаяся форма RPC в углу:
- использован `details` popover-блок в правом верхнем углу;
- перенесена существующая форма подключения к RPC (`rpc-config-form`).

3. Обновлены стили в `styles.css`:
- адаптивная двухколоночная сетка;
- визуальное разделение правой панели на верх/низ;
- оформление action-карточек;
- фиксированный popover в углу;
- мобильная адаптация (колонки складываются в одну).

## Результат
Интерфейс стал соответствовать сценарию использования: ввод данных слева, операционные действия справа, история отдельно снизу, быстрый доступ к RPC-конфигу из угла экрана.
